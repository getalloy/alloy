What is alloy?

alloy is a set of tools for the modern web built with Metalsmith. Think of it as a really nice boilerplate for Metalsmith with an opinionated, streamlined approach. alloy gives you everything you need to get up and running with Metalsmith and deploy static sites with ease. It is the result of things learned building and deploying websites for Circle with Disney.
